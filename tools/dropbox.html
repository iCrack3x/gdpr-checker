Under GDPR, any tool that processes EU resident data must:

  • Have a valid legal basis for processing
  • Provide transparency about data usage
  • Ensure data is only transferred to adequate countries (or with Standard Contractual Clauses)
  • Respond to data subject requests (access, deletion, portability)

Dropbox is based in or sends data to the US, which currently lacks an adequacy decision following Schrems II. This makes compliance complex.
